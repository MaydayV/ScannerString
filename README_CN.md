# ScannerString

一个强大的 Swift 字符串扫描工具，帮助您分析和管理 Swift 项目中的字符串。

## 功能特点

- 扫描 Swift 源代码文件中的字符串字面量和正则表达式
- 识别本地化字符串（NSLocalizedString）
- 为每个字符串提供详细的位置信息（文件、行号、列号）
- 并发文件扫描以提高性能
- JSON，csv，.strings，.xcstrings 输出格式，便于与其他工具集成
- 自动排除常见目录（如 Pods、Carthage、Tests 等）

## 系统要求

- Swift 5.0 或更高版本
- macOS 10.15 或更高版本
